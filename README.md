# THICC OR FAT: Server

## Purpose
  * To keep track of the poll of whether Trump is THICC or FAT

## Dev
 * `npm install`
 * to run and watch : `npm run dev`

## Tech
  * Node.js
  * nodemon
  * express
  * mongoose
  * bluebird
  * underscore.js
  * dotenv
  * Heroku + Mongolab add-on
  * body-parser
